## Key for Reading This Table

- **HR** = Hazard Ratio (HR < 1.0 means the intervention reduces risk)
- **eOR** = Equivalent Odds Ratio (converted for cross-study comparability)
- **RR** = Risk Ratio
- **95% CI** = 95% Confidence Interval (if the interval crosses 1.0, the result is not statistically significant)
- **NS** = Not statistically significant (p > 0.05)
- **MACE** = Major Adverse Cardiovascular Events (typically: CV death + non-fatal MI + non-fatal stroke)
- **GRADE** = Grading of Recommendations, Assessment, Development and Evaluation (high/moderate/low/very low certainty)
- **Bold** values indicate statistically significant results
